Postgraduate Dissertation

Project Overview

This is my postgraduate dissertation. For this project I created a game in Unreal Engine. This is a template of the kind of game that could be created to help research into cognitive bias. I created an escape room game where buttons would have unexpected outputs. The player would be trying to escape the room in the time limit. There are menus and the ability to restart levels as well as a small amount of NPC AI.

Undergraduate Dissertation

Project Overview

This is my undergraduate disseration. For this project I created a hand tracking tool in Unity using OpenCV and Python sockets. This project was based on research into games for rehabilitation purposes. The tool uses a webcam as an input and allows the player to move the ingame hand using this. They are able to interact with buttons in the game world. When all the buttons have been pressed the level complete screen will display with the users final time.
