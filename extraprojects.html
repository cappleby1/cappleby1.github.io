A Level Coursework

For my A Level coursework I created a basic platformer game with one level that could be completed by the player by reaching the goal. This was the first game development project I undertook. From this I learnt a lot about using C# as a language and how game engines work and can be used.



Technologies & skills this project included were:

  • Began to learn C# and unity.
  • First bit of game engineering work I ever completed.
  • Completed the project without the use of an IDE (Notepad++).

XR Unity App Project

For this project I created a mobile app that allows the user to scan a QR code and spawn a box that can be interacted with. I used ZXing for this project with unity. It was my first time working with XR and I found it very interesting to learn about.



Technologies & skills this project included were:

  • First time using ZXing
